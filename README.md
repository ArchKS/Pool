# Pool

1. 给我安装React+Tailwind+rsbuild的前端Web开发方案

我要用React+tailwind生成一个网页，展示我买的股票的相关数据：
1、左侧导航栏有标的名称，每个标的包含以下字段：名称、分类、标签、评级、完成度、最近更新日期；相同的分类用相同的颜色标识；可以按照评级、完成度排序；可以按照分类归类，相同的分类在一起且分类中子项多的在上面，比如有色金属分类有10个标的，白酒有3个标的，水泥有5个标的，则按照分类归类后，有色金属第一，水泥第二，白酒第三，且分类里的标的按照评级降序排列；
2、右侧内容显示，点击分类显示对应的内容，主要是echarts图表和文字/表格，用户可以新建表格和文字，且支持导出成json，这个json可以放在本地，供右侧内容显示区域加载
